### ABC154

# B - I miss you...

  [問題はこちら](https://atcoder.jp/contests/abc154/tasks/abc154_b)


- 発想<br>
  S の文字列数だけ x を出力する。 


- コード（C++）

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;


  int main() {

    string S;
    cin >> S;

    for (int i = 0; i < S.size(); i++) {
      cout << "x";
    }

    cout << endl;

    return 0;

  }
  ```
