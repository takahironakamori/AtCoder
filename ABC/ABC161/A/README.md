### ABC161

# A - ABC Swap

  [問題はこちら](https://atcoder.jp/contests/abc161/tasks/abc161_a)


## 発想

  Z, X, Yの順番で出力する。


## コード（C++）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {

  int X, Y, Z;
  cin >> X >> Y >> Z;

  cout << Z << " " << X << " " << Y << endl;

  return 0;

}
```
