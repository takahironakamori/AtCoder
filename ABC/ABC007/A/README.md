  [問題はこちら](https://atcoder.jp/contests/abc007/tasks/abc007_1)

- 発想<br>
  n - 1 が答え<br>

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;

  int main() {

  }
  ```

  ```java
  import java.util.*;
  public class Main {
    public static void main(String[] args){
      Scanner sc = new Scanner(System.in);
      int n = sc.nextInt();
      System.out.println(n - 1);
    }
  }
  ```
